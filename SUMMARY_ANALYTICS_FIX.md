# Исправление: Аналитика не отображается в админ-панели

## Краткое описание
Исправлена проблема, когда аналитика пользователя не отображалась в админ-панели, хотя данные существовали в базе данных.

## Причина
Несоответствие типов ID: в URL использовался внутренний `user.id`, а API ожидал `sso_uid`.

## Изменения
- ✅ `src/app/admin/users/page.tsx` - использование `sso_uid` в URL
- ✅ `src/app/api/admin/users/[userId]/entries/route.ts` - правильная конвертация ID
- ✅ `src/app/api/admin/users/[userId]/route.ts` - использование `sso_uid`
- ✅ `src/app/api/admin/users/[userId]/analytics/generate/route.ts` - правильная конвертация ID

## Результат
✅ Сборка успешна
✅ TypeScript проверка пройдена
✅ Аналитика теперь корректно отображается в админ-панели
