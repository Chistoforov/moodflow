# ⚡ БЫСТРОЕ ИСПРАВЛЕНИЕ ОШИБКИ 500

## Проблема
При открытии сайта появляется ошибка:
```
500: INTERNAL_SERVER_ERROR
Code: MIDDLEWARE_INVOCATION_FAILED
```

## Причина
На Vercel не установлены переменные окружения.

## Решение за 3 шага

### Шаг 1: Найдите ваши ключи Supabase

1. Откройте https://supabase.com
2. Выберите ваш проект
3. **Project Settings** → **API**
4. Скопируйте:
   - **Project URL** (например: `https://abc123.supabase.co`)
   - **anon public** (длинный токен, начинается с `eyJ...`)
   - **service_role** (еще один длинный токен)

### Шаг 2: Добавьте их на Vercel

1. Откройте https://vercel.com
2. Выберите ваш проект
3. **Settings** → **Environment Variables**
4. Добавьте **3 переменные** (для каждой выберите Production, Preview, Development):

```
Имя: NEXT_PUBLIC_SUPABASE_URL
Значение: https://abc123.supabase.co (ваш URL)
```

```
Имя: NEXT_PUBLIC_SUPABASE_ANON_KEY
Значение: eyJ... (ваш anon токен)
```

```
Имя: SUPABASE_SERVICE_ROLE_KEY
Значение: eyJ... (ваш service_role токен)
```

### Шаг 3: Redeploy

1. **Deployments** → кликните на последний деплой
2. Три точки (⋯) → **Redeploy**
3. Дождитесь завершения (2-3 минуты)
4. Откройте ваш сайт - должно работать! ✅

## Дополнительные переменные (добавьте позже)

Для полной функциональности добавьте также:

- `OPENAI_API_KEY` - для AI рекомендаций
- `PERPLEXITY_API_KEY` - для аналитики
- `TELEGRAM_BOT_TOKEN` - для уведомлений
- `CRON_SECRET` - любая строка (например: `secret123`)
