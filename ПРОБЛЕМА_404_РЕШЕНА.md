## Возможные ошибки

### 400 Bad Request - Not enough entries
**Причина**: У пользователя меньше 3 записей в текущем месяце  
**Решение**: Выберите другого пользователя или добавьте записи

### 403 Forbidden - Admin access required
**Причина**: Текущий пользователь не является админом  
**Решение**: Выполните SQL в Supabase Dashboard:

```sql
-- Проверьте вашего пользователя
SELECT u.email, p.role, p.active
FROM users u
LEFT JOIN psychologists p ON p.user_id = u.sso_uid
WHERE u.email = 'ваш_email@example.com';

-- Если роль не admin, обновите:
UPDATE psychologists
SET role = 'admin', active = true
WHERE user_id = (
  SELECT sso_uid FROM users WHERE email = 'ваш_email@example.com'
);
```

### 500 Internal Server Error - RLS policy violation
**Причина**: Не применены миграции для RLS политик  
**Решение**: Примените миграции 024 и 025

См. файлы:
- `supabase/migrations/024_create_monthly_analytics.sql`
- `supabase/migrations/025_fix_monthly_analytics_admin_insert.sql`

Или используйте инструкцию: `ПРИМЕНИТЬ_МИГРАЦИИ_024_И_025.md`
