# 🔍 Как проверить логи в Vercel

## СРОЧНО - Проверьте логи сервера:

1. **Откройте Vercel Dashboard**
   - https://vercel.com/dashboard
   - Выберите проект MoodFlow

2. **Перейдите в раздел Logs**
   - В верхнем меню: "Logs"
   - Или прямая ссылка к последнему деплою

3. **Запустите анализ снова** в админ-панели
   - Откройте вашу админ-панель
   - Нажмите "Анализ" на любом пользователе

4. **Обновите логи в Vercel** (они обновляются в реальном времени)
   - Вы должны увидеть сообщения с эмоджи:
     - 🔍 Attempting to fetch user: ...
     - 🔍 Current auth user: ...
     - 📊 User query result: ...
     - ❌ User not found: ...

5. **Сделайте скриншот логов** и покажите мне

## Что искать в логах:

- `userError` - объект с деталями ошибки Supabase
- Может быть ошибка RLS: "row-level security policy"
- Или ошибка аутентификации
